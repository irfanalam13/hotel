# PHASE_03_FRONTEND_UI.md

Build a world-class hotel management frontend.

Tech Stack:

* Next.js 15 App Router
* TypeScript
* Tailwind CSS
* shadcn/ui
* Framer Motion
* Sonner Toast
* React Hook Form
* Zod
* TanStack Query

Design Goals:

* Premium SaaS quality
* Modern dashboard
* Smooth animations
* Fast navigation
* Mobile responsive

Pages:

* Login
* Dashboard
* Properties
* Rooms
* Reservations
* Guests
* Billing
* Housekeeping
* Inventory
* Reports
* Settings

UI Requirements:

* Animated sidebar
* Animated cards
* Skeleton loaders
* Toast notifications
* Empty states
* Error states
* Success states

User Experience:

* Framer Motion page transitions
* Optimistic updates
* Real-time notifications
* Keyboard shortcuts
* Command palette

Design Inspiration:

* Linear
* Stripe Dashboard
* Notion
* Vercel Dashboard

Create reusable components.

Follow Feature-Based Architecture.

All forms must use React Hook Form + Zod.
