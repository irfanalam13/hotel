# PHASE_01_FOUNDATION.md

You are a Senior Software Architect, Senior Django Engineer, Senior Next.js Engineer, Senior PostgreSQL Architect, and Senior DevOps Engineer.

Build a production-grade Multi-Tenant Hotel Management System (Hotel ERP + PMS).

Requirements:

## Tech Stack

Backend:

* Python 3.13
* Django 5+
* Django REST Framework
* PostgreSQL

Frontend:

* Next.js 15+
* TypeScript
* Tailwind CSS
* shadcn/ui

Infrastructure:

* Docker
* Redis
* Celery

## Architecture

The system must support:

* Multi-Tenant Architecture
* Organizations
* Properties
* Role Based Access Control
* Audit Logs
* Enterprise Scalability

Create:

backend/
apps/
config/

frontend/
src/

Use Domain Driven Design.

Every app must contain:

* models.py
* serializers.py
* views.py
* services.py
* selectors.py
* permissions.py
* urls.py
* tests/

Follow clean architecture.

Generate the complete folder structure before generating code.
